# Scaffold with CLI

## C\#
```
dotnet new console -o hello
cd hello
dotnet run
```

## Dart
```
dart create -t cli hello
cd hello
dart run bin\hello.dart
```

## flutter
```
mkdir hello
cd hello
flutter create .
flutter pub get
flutter run
flutter run --release

flutter build apk
flutter install
```

## Rust
```
cargo new hello -- bin
cd hello
cargo run
```
