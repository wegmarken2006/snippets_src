# Scaffold with CLI

## C\#
```
dotnet new console -o hello
cd hello
dotnet run
```

## Dart
```
pub global activate stagehand
mkdir hello
cd hello
stagehand console-full
pub get
dart bin/main.dart
```

## flutter
```
mkdir hello
cd hello
flutter create .
flutter pub get
flutter run

flutter build apk
flutter install
```

## Rust
```
cargo new hello -- bin
cd hello
cargo run
```
